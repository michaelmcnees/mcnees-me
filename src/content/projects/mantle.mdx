Mantle is a workflow automation tool built for cloud engineers who already live in Terraform and want something with the same shape — validate, plan, apply, run — for AI and general automation. The top goals are security and ease of use: BYOK, single Go binary, Postgres-only state, and a threat model that assumes your workflows touch sensitive things. I built it because the existing AI workflow tools don't take security seriously enough.

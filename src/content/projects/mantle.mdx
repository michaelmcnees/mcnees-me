Mantle takes the IaC lifecycle I already trust — validate, plan, apply, run — and points it at AI workflow automation. It's a single Go binary with BYOK, Postgres-only state, and crash recovery that actually works. I built it because every workflow tool I tried either pretended AI didn't exist or drowned in it.
